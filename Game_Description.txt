Игра "Catch the Enemies" аркадного типа, вид сверху.

Главный герой (ГГ) ловит противников коробкой. 3 противника: движущийся по прямой, движущийся зигзагом и движущийся по спирали.

ГГ изображается зеленым квадратом, коробка изображается красным квадратом, расположенным рядом с ГГ, а противники изображаются синими квадратами.
В начале игры противники не движутся в сторону ГГ.
ГГ движется при помощи клавиш W, A, S, D. Также есть возможность смещать коробку при помощи клавиш Стрелка влево, Стрелка вправо, Стрелка вниз, Стрелка вверх. 
Далее если происходит столкновение противника с коробкой, счет игры увеличивается на 1 очко, 
противник исчезает и создается новый противник одного из трех типов случайным образом. 
Приэтом новый противник уже имеет компоненту скорости, направленную к ГГ (скорость пропорциональна счету).
Текущий счет игры выводится в консоль, если происходит столкновени ГГ с противником, игра заканчивается.

Запуск игры:

1. Распаковать проект.
2. Открыть проект в Visual Studio
3. Нажать Локальный отладчик Windows
